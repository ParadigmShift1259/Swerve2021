15 ## Running Paths
16 Team 1259 is currently using Team 3015's PathPlanner software to generate swerve paths.
17 PathPlanner generates a C++ array which is converted into an frc::Trajectory, an object that carries desired states via timestamps for our Auto runs.
18 ### Setup
19 - Download PathPlanner latest release (2021 uses **v1.6**) executable for Windows `pathplanner-win.exe` from [**here**](https://github.com/mjansen4857/pathplanner/releases/tag/v1.6.0) and install the .exe
20 - Read through the [**Path README**](PathREADME.md) (Copied from PathPlanner v1.6)
21 - Get example swerve paths for the 2021 robot [**here**](https://github.com/ParadigmShift1259/FRC_Robot_2021/tree/paths)
22 ### Config
23 - Open up PathPlanner
24 - In the top left settings menu, set the following:
25 
26 | Name | Value |
27 |----------------------- |----------------------------- |
28 | Team Number | 1259 |
29 | RoboRIO Path Location | (Don't change) |
30 | Units | Metric (will be in meters) |
31 | Game Year | Your year |
32 | Max Velocity | Robot dependent - calculate |
33 | Max Acceleration | Robot dependent - calculate |
34 | Wheelbase Width | Robot dependent - calculate |
35 | Robot Length | Robot dependent - calculate |
36 | Time Step | 0.02 |
37 | Drive Train | Holonomic |
38 
39 <br/>
40 
41 - In the bottom left menu, click Generate Path (ctrl+G) to create the C++ array. Set the following:
42 
43 | Name | Value |
44 |----------------- |------------------------- |
45 | Output Type | C++ array |
46 | Path Name | Name of autonomous path |
47 | Output Format | t,v,a,X,Y,hh |
48 | Reversed Output | false (Off) |
49 | Split Path | false (Off) |
50 
51 <br/>
52 
53 - You should now be able to generate example paths from 2021 or your own paths!
